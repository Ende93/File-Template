# File Template


An visual studio code extension for creating file from templates automatic.   
一个自动生成文件模板的 VScode 扩展。



## How to use  

1. Open a file by VScode.  
2. `Ctrl + Shift + P`.  
3. Input `Tmpl: Create Template` and press `Enter`. 


# Supported Type


- javascript
- html
- css
- php
- python
- ruby
- xml
